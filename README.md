# 个人主页

这是我的个人主页项目，使用 HTML 和 CSS 构建的响应式网站。（现在还处于demo阶段）

## 功能特点

- 响应式设计，适配各种设备
- 现代化的UI界面
- 技能展示
- 项目展示
- 社交媒体链接

## 自定义

可以通过修改 `styles/main.css` 中的 CSS 变量来自定义网站的主题颜色：

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #3b82f6;
    --text-color: #1f2937;
    --bg-color: #ffffff;
    --section-bg: #f3f4f6;
}
```
